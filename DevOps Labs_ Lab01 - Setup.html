Lab 01: Setup

CMPS4131 Software Engineering: Development Environment & DevOps Setup workflow using VS Code, GitHub Enterprise (UB), and Docker

Overview

The aim of this lab is to set up a professional, flexible software engineering and DevOps development environment that you will use throughout CMPS4131 Software Engineering.

This lab is technology-agnostic. You are not restricted to a specific programming language or framework. Your project may be a web application, a JavaScript / Node.js project, a Java project, a Python application, or any other instructor-approved software project.

What matters is how you work, not which language you use. This lab focuses on tools and workflows used across modern software engineering roles.

Tools Used in This Module

  • Visual Studio Code (VS Code) code editor
  • Git version control
  • GitHub Enterprise (UB Account) remote repository hosting
  • Docker containerization platform

These tools form a modern DevOps-enabled development environment and will be used for coursework, projects, and assessment.

Learning Objectives

After completing this lab, you will be able to:

  • Set up a development workspace using VS Code
  • Use Git for local version control
  • Create and manage repositories using GitHub Enterprise
  • Write clear project documentation using Markdown
  • Run and manage Docker containers
  • Create a Dockerfile for your project
  • Deploy an application into a containerized environment
